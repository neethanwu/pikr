## Design Context

### Users
Developers using terminal-based AI coding agents (Claude Code, Codex, OpenCode, Amp). They need to visually identify UI elements and give structured context to their agent. They're technical, fast-moving, and impatient with friction. pikr is a tool they use alongside their editor and terminal — it should feel like a natural extension of their workflow, not a separate app.

### Brand Personality
**Fast, minimal, sharp.**

pikr has a point of view. It's opinionated about how element picking should work and it doesn't apologize for it. The coral accent is the single brand moment — everything else stays quiet.

### Aesthetic Direction
- **References:** Arc (distinctive, opinionated), Figma (design-forward, functional)
- **Anti-references:** Generic SaaS/Bootstrap (no blue-gray corporate), over-designed tools (no gradients, neon, purple-on-dark). Not generic AND not heavy.
- **Theme:** The overlay runs on OTHER people's pages. It must look good on any background — light, dark, colorful. Glassmorphism (backdrop-filter blur) + adaptive borders handle this.

### Design Tokens (established)

| Token | Value | Role |
|---|---|---|
| Accent | `#ff6b56` (coral) | Brand color, inspect dot, wordmark in CLI |
| Success | `#a3e635` (aurora lime) | Capture confirmation, glow ring |
| Surface dark | `#1c1917` (stone-900) | Inspect mode banner |
| Surface light | `rgba(255,252,249,0.92)` (warm cream) | Browse mode banner, toast |
| Text dark | `#292524` (stone-800) | Primary text on light |
| Text light | `#fafaf9` (stone-50) | Primary text on dark |
| Radius | 12px (pills), 8px (small), 3px (highlight) | |
| Font | System sans-serif | UI chrome |
| Mono | `ui-monospace, "SF Mono", "Cascadia Mono"...` | Labels, element info, wordmark hint |
| Ease | `cubic-bezier(0.16, 1, 0.3, 1)` (ease-out-expo) | All animations |
| Shadow | `rgba(28,25,23,...)` | Warm, not cold black |

### Design Principles

1. **Invisible until needed.** The overlay should enhance the page, not compete with it. Minimum visual footprint — compact pill, auto-snap to edges, no unnecessary chrome.

2. **Instant feedback.** Every interaction gets immediate response. Highlight snaps (no transitions on position), capture pulse is <100ms, toast appears without delay. Speed is the primary UX metric.

3. **Works on any page.** The overlay runs on other people's sites — dark themes, light themes, colorful backgrounds, dense layouts. Adaptive borders, glassmorphism, and warm neutrals ensure visibility everywhere.

4. **One brand moment.** Coral `#ff6b56` is the only color that says "pikr." Everything else is neutral. The dot, the wordmark in CLI, the highlight border — coral is the thread. Don't dilute it with other accent colors.

5. **Respect reduced motion.** All animations collapse to instant when `prefers-reduced-motion` is set. No exceptions. The tool must be fully functional without any motion.
